          Todo App User Guide

          1. Managing Your Workspace

          Your data is organized into Workspaces. Use the navigation menu (Mobile: tap the toggle menu to reveal) to switch, add, or update workspaces.

          2. Task Management & Status Rules

          Every task has a status that dictates its interactivity:

          • Pending:

            Active tasks that you can mark as "Done."

          • Done (Past Due):

            If a task is completed and the due date has passed, it is locked.

          • Undone:

            Tasks that pass their due date without being marked as "Done" are automatically flagged as "Undone" and are locked to prevent accidental status changes.

          Note: If a task is locked (Done or Undone), you can still use the Edit button to update the task details or date. Any successful update will reset the task status to Pending, allowing you to interact with it normally again.

          3. Mobile vs. Desktop Navigation

          While the interface is similar, please note the following difference:

          Mobile Users:

          Because screen space is limited, the primary action controls for tasks are contained within a secondary view.

          Note: You must click the dedicated Action Button located near the table row to expand and reveal the hidden Edit and Delete controls.

          4. Productivity Features

          • Drag-and-Drop:

            You can manually reorder your tasks by clicking and holding any row, then sliding it to your desired position.

          • Search:

            Use the search input to quickly locate specific workspaces.

          • Keyboard Shortcuts:

            Press Enter while focused on any input field (task name, date, or search) to submit your changes instantly.

          ⚠️ Important Warnings

          Note: This application uses your browser's local storage. Clearing your browser cache or site data will permanently delete all your tasks and workspace configurations.

          Note: Task dates cannot be set to the past. The system will alert you and prevent the action if an invalid date is submitted.
